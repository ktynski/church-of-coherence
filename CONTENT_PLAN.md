## Copy Principles

1. **Hook emotionally, satisfy intellectually**: Start with the *why* (meaning crisis), then deliver the *how* (Coherence structure)
2. **Rhythm**: Vary sentence length. Use fragments for emphasis. Build momentum.
3. **Avoid jargon initially**: Introduce terms gradually with context
4. **Use metaphor**: "Grace is a contraction" → "Grace is like gravity, pulling distortion back toward center"
5. **Anticipate resistance**: Address "this sounds crazy" before reader thinks it
6. **Concrete before abstract**: Start with examples, then formalize
7. **Story arc**: Every section should have narrative flow
