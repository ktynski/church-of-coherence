# Church of Coherence

A new church built on **the Coherence framework**, with the goal of uniting all world religions through deeper understanding.

## Vision

The Coherence framework offers a formal understanding where:

- **Grace** — the transformation that makes closure possible — unwinds distortion without erasing the structure that makes manifestation real
- **Coherence** — not uniformity, but the capacity for different perspectives to recognize one another
- **The golden ratio φ** — emerges as a universal threshold: the boundary between productive differentiation and terminal rupture

The church seeks to translate these structural insights into a shared language that honors every tradition while revealing the common ground beneath apparent contradiction.

## Core Principles (from the Coherence framework)

1. **Closure without collapse** — Reality requires structure (χ = 1), but structure requires differentiation. The tension is generative, not fatal.

2. **Grace recovers** — Not all distortion is evil. Evil is specifically distortion that turns against reintegration. Grace is a strict contraction: single ruptures never cause permanent exile.

3. **Witness** — Coherence is not merely internal; it is legible. The Spirit is the coherence witness — the solver, the detector, the one who recognizes rank-collapse.

4. **Unification through depth** — Religions are not to be flattened into one. They are to be understood at a depth where their differences become complementary rather than contradictory.

## Getting Started

Serve the folder with any static file server:

```bash
cd church-website
python -m http.server 8000
# or: npx serve .
```

Then open `http://localhost:8000` in your browser.
